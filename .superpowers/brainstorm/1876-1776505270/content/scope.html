How far should the theme go?

The Login X brand has two personalities — neon dark mode and clean light mode. Which scope fits MovieNexus?

A

Dark mode only — neon treatment

Keep MovieNexus dark-only (it's a media app, dark suits it). Replace the current indigo accent with Electric Purple + Neon Cyan. Add glow effects on poster cards and the nav logo. No light mode needed.

B

Full dual-mode — match Login X exactly

Add both dark (neon cyberpunk) and light (clean Stripe-like) modes with a theme toggle. Same palette, same feel as Login X. More work but a consistent brand across all your apps.

C

Dark mode + subtle brand colors only

Apply the brand palette (purple/cyan) but keep animations and glow effects minimal — just color tokens swapped. Faster to implement, easier to maintain.
